### 🔐 PPRL Machine Learning Privacy Risk Assessment
Implemented a black-box Membership Inference Attack (MIA) using shadow models to evaluate privacy leakage in a machine learning system.

- Designed a shadow model pipeline with embedding splitting
- Applied ROC-based threshold optimization
- Evaluated false positive rates under multiple attack strategies
- Analyzed correlation between overfitting and privacy risk

📌 Key Outcome:
Higher overfitting levels significantly increased membership inference vulnerability.
