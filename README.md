### 🔐 Machine Learning Privacy Risk Assessment (MIA)
Implemented a black-box Membership Inference Attack to evaluate privacy leakage in a machine learning model.

- Designed shadow model pipeline
- Extracted confidence-based features (entropy, probability gap)
- Evaluated attack performance using ROC/AUC
- Analyzed relationship between overfitting and privacy risk

👉 **Key Outcome**
- Verified low privacy leakage (AUC ≈ 0.48) and identified conditions where risk increases
